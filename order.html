Способы оплаты
- безналичная оплата на расчетный счет с выделением НДС 20% - только для ООО и ИП
- безналичная оплата на расчетный счет БЕЗ выделения НДС - только для ООО и ИП
- наличный расчет - только на условиях САМОВЫВОЗА ( адрес склада Новосибирск, ул. Мира-62, с 7,00 до 16,00 в рабочие дни )
- возможны иные способы оплаты для Частных лиц по договоренности
